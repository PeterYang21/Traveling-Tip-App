# Travelling Tip App
## Description
Provide tips for travelling to various cities by showing their distinct features through views of locally traditional plates and places of interest. Users can create accounts, create/update profiles and favorite places interested in for further travelling plans while navigating app. App data including usersâ€™ information is managed by SQLite database
